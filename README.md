# MERN NoteMaker App

A full-stack Note Making App using MongoDB, Express.js, React.js (Vite), and Node.js.

## Features
- User Authentication (Login/Signup)
- Create, Read, Delete Notes
- Dark Mode Toggle
- Search Notes
- Responsive & Stylish UI

## Stack
- MongoDB
- Express.js
- React.js (Vite)
- Node.js
